# Map Generator

A simple GUI app to generate, fiddle with and save Texture Maps.
Built with Qt framework.

Implemented Features:

- none :)

planned features:

- White Noise
- Simplex Noise
- Perlin Noise
- Cellular Automata
- Voronoi

## Building

I was working on linux (5.15.6-arch2-1), from the terminal (no Qt Creator)

```
qmake -project QT+=widgets
qmake
make
```
